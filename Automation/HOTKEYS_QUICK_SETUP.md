## Быстрая настройка хоткея для External Tool

1. Откройте Settings → Keymap
2. External Tools → HuyEngine Build Menu
3. ПКМ → Add Keyboard Shortcut → нажмите `Ctrl+NumPad1`

Если у вас отсутствует External Tool — настройте его по инструкции выше (CLION_HOTKEYS.md).

## Альтернативные комбинации

- Ctrl+Shift+B
- Alt+B
- Ctrl+1 (если NumPad не работает)
